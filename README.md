# SOHMNA — Sample Intelligence

A sample DNA explorer powered by Claude AI.

## What it does
- Drag & drop any audio file to analyze its DNA (BPM, key, era, genre)
- Finds the original source it sampled from
- Surfaces sibling tracks that sampled the same original
- Genre browser across Funk, Soul, Jazz, Breaks, Reggae, Disco
- AI Crate Digger chat for deep sample research
- Save and revisit past analyses

## Files
- `index.html` — the entire app (single file)

## Setup
1. Open `index.html` in any browser — it works offline
2. Your Anthropic API key is pre-loaded
3. Drop a track and hit Analyze

## Deploy updates
Drag the `sohmna` folder onto https://app.netlify.com/drop to redeploy
